# Pastebin-Lite Backend

This is the backend service for the Pastebin-Lite application.

## Tech Stack
- Node.js
- Express.js
- MongoDB (Atlas)
- Mongoose

## Features
- Create a paste
- Fetch a paste via API
- View paste as HTML
- Supports TTL (expiry) and max view limits
- Health check endpoint

## Environment Variables

Create a `.env` file in the backend root

## Install & Run

npm install
npm run dev
